# Gerador de Senhas

Aplicação web para geração de senhas seguras com opções personalizadas, desenvolvida com HTML, CSS e JavaScript.

## Acesso ao projeto
🔗 [Clique aqui para acessar](https://phmbezerra.github.io/gerador-de-senhas/)

## Funcionalidades
- Gerar senhas automaticamente
- Escolher o tamanho da senha
- Incluir letras maiúsculas
- Incluir números
- Incluir símbolos
- Copiar senha gerada com um clique

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Objetivo do projeto
Este projeto foi desenvolvido para praticar:
- manipulação de formulários e inputs
- geração dinâmica de conteúdo
- lógica de combinação de caracteres
- interação com a área de transferência
- construção de interfaces úteis e funcionais

## Como executar
1. Baixe ou clone este repositório
2. Abra a pasta do projeto
3. Execute o arquivo `index.html` no navegador

## Estrutura do projeto
index.html

README.md

## Aprendizados
Durante o desenvolvimento deste projeto, foram praticados conceitos como:
- manipulação do DOM
- uso de eventos em JavaScript
- atualização em tempo real de valores na interface
- lógica de geração aleatória
- interação com navigator.clipboard
