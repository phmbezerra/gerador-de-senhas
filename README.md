## Sobre o projeto

O **Gerador de Senhas** é uma aplicação web feita para criar senhas fortes de forma simples e rápida, permitindo personalização do tamanho e dos tipos de caracteres utilizados.

Além da funcionalidade principal, o projeto também foi pensado para ter uma apresentação visual mais marcante, com uma interface mais fofa, estilizada e autoral, utilizando ilustrações personalizadas para reforçar o tema de **segurança, sigilo e proteção de dados**.

Essa versão atualizada foi desenvolvida para unir:
- funcionalidade;
- identidade visual;
- melhor experiência do usuário;
- valor estético para portfólio.

## Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar e demonstrar conceitos importantes do desenvolvimento front-end, incluindo:

- estruturação semântica com HTML;
- estilização moderna com CSS;
- responsividade;
- manipulação do DOM com JavaScript;
- geração dinâmica de conteúdo;
- interação com o usuário;

- organização visual mais criativa e diferenciada para portfólio.
